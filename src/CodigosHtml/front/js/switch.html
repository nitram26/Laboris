Switch

  • Teoría

    switch es una estructura de control que permite ejecutar diferentes bloques de código según el valor de una expresión.
    Es útil cuando tienes múltiples condiciones que derivan de la misma variable o expresión.

    switch (expresión) {
    case valor1:
    // código a ejecutar si expresión === valor1
    break;
    case valor2:
    // código a ejecutar si expresión === valor2
    break;
    case valorN:
    // código a ejecutar si expresión === valorN
    break;
    default:
    // código a ejecutar si ninguno de los casos coincide
    }

    expresion: normalmente una variable o una expresión cuyo valor se compara con los case.
    case valor: cada una de las posibles coincidencias. La comparación utiliza estrict equality (===).
    break: detiene la ejecución y salta fuera del switch. Si se omite, se ejecutarán los siguientes case (caída o “fall-through”).
    default: opcional. Se ejecuta si ninguna coincidencia se cumple.



  1. Dado un puntaje numérico score, imprime la categoría de rendimiento:

    "Excelente" para score >= 90
    "Bueno" para score >= 75
    "Regular" para score >= 60
    "Bajo" para cualquier otro valor


    const score = /* valor numérico */;
    switch (true) {
    case score >= 90:
    console.log('Excelente');
    break;
    case score >= 75:
    console.log('Bueno');
    break;
    case score >= 60:
    console.log('Regular');
    break;
    default:
    console.log('Bajo');
    }

  2. Damos una variable comando que puede ser 'iniciar', 'parar', 'reiniciar' o 'status'. Ejecuta la función correspondiente para cada comando. Si no es ninguno de esos, imprime 'Comando desconocido'.

    const comando = /* valor de comando */;
    const accionar = {
    iniciar: () => console.log('Arrancando...'),
    parar: () => console.log('Parando...'),
    reiniciar: () => console.log('Reiniciando...'),

    if (accionar) accionar();

  3. Tenemos una variable código que puede ser 'A1', 'B2', 'C3' o cualquier otro. Por cada código, imprime el tipo de carnet:

    Cualquier otro → "Carnet desconocido"

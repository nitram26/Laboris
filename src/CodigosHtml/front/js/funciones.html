  1. TIPOS DE FUNCIONES
    1. Función Declarada

      2. Tiene la forma :
      function nombreDeFunction ( ) {

      //código de la función

      return valor;
      }

      Se llama usando: nombreDeFunction ()

    2. ¿Por qué es importante return?
      Permite que la función devuelva un resultado que puedas usar después.
      La función deja de ejecutarse en el momento en que encuentra una instrucción return.
      Sin return, la función devuelve automáticamente undefined.

      console.log es una función que imprime en la consola del navegador.
      console.log no devuelve un valor, solo muestra información en la consola.
    3. Función Expresada o Anónima

      Tiene la forma:
      const nombreDeFunction = function ( ) {

      //código de la función

      }

      Se llama usando: nombreDeFunction ()

    4. Con Parámetros

      Tiene la forma:
      function nombreDeFunction (parametro1, parametro2) {
      }

      Se llama usando: nombreDeFunction (valor1, valor2)

      Se usa también en el formato de función Anónima

      Para asignar valores por defecto a los parámetros, se puede hacer de la siguiente manera:
      function nombreDeFunction (parametro1 = valorPorDefecto1, parametro2 = valorPorDefecto2) {
      }

    }

    const array1 = [1, 2, 3, 4, 5];
    console.log(sumarArreglo(array1));


    /*
    array.reduce(funcion, valorInicial)
    funcion: Una función que toma dos argumentos:
    acumulador (el valor acumulado en las iteraciones)
    elemento actual (el elemento en la posición actual del array)
    valorInicial (opcional): Un valor inicial para el acumulador. Si no se proporciona, el primer elemento del array se usa como valor inicial y la iteración empieza desde el segundo elemento.
    ¿Qué devuelve?
    reduce devuelve un único valor, que es el resultado final tras aplicar la función a todo el array.

    Ejemplo sencillo: sumar todos los números en un array
    Supongamos que tienes este array:

    js
    const numeros = [1, 2, 3, 4, 5];
    Y quieres sumar todos los números. Puedes usar reduce así:

    js
    const suma = numeros.reduce((acumulador, valorActual) => {
    return acumulador + valorActual;
    }, 0);
    ¿Qué pasa aquí?

    El acumulador empieza con el valor inicial 0 (que pasamos como segundo argumento a reduce).
    Para cada elemento en el array, la función se ejecuta:
    En la primera iteración: acumulador es 0, valorActual es 1
    Devuelve 0 + 1 = 1
    En la segunda: acumulador es 1, valorActual es 2
    Devuelve 1 + 2 = 3
    En la tercera: acumulador es 3, valorActual es 3
    Devuelve 3 + 3 = 6
    Y así sucesivamente hasta recorrer todo el array.
    Al final, suma tendrá el valor 15.

    */

  18. Escribe una función llamada ordenarArreglo que reciba un arreglo de números y lo ordene de menor a mayor

    const ordenarArreglo =function (arreglo) {

    return arreglo.sort((a, b) => a - b);
    }

    let sol = ordenarArreglo([5, 3, 8, 1, 2]);
    console.log(sol);


    /* Usamos la función de comparación para ordenar números correctamente La expresión (a, b) => a - b es una función anónima en JavaScript, conocida como función flecha. Esta función se usa como función de comparación en el método sort() para ordenar números correctamente. Aquí te lo explico paso a paso: 1. La sintaxis (a, b) => a - b (a, b): indica que la función recibe dos parámetros, llamados a y b. En el contexto de sort(), estos representan dos elementos del arreglo que se están comparando en cada paso. =>: es la flecha que indica que lo que sigue es el cuerpo o la expresión de la función. a - b: es la expresión que se evalúa y que determina el orden. 2. Cómo funciona en sort() La función de comparación debe devolver: Un número negativo si a debe ir antes que b. Cero, si son iguales. Un número positivo si a debe ir después que b. Al usar a - b: Si a es menor que b, la resta a - b será negativa → a antes que b. Si a y b son iguales, la resta será 0 → su orden no cambia. Si a es mayor que b, la resta será positiva → a después de b. Ejemplo con números Supón que tienes el arreglo: [3, 1, 4, 2] Al ordenar con: js arreglo.sort((a, b) => a - b); El proceso será: Compara 3 y 1: 3 - 1 = 2 (positivo) → 3 después de 1. Compara 3 y 4: 3 - 4 = -1 (negativo) → 3 antes de 4. Y así sucesivamente, hasta que el arreglo quede ordenado: [1, 2, 3, 4]. */

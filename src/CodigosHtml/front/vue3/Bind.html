  1. Cambio de color de Etiqueta

    Hacer una mensaje que al pulsar un botón cambie de color y después de 2 segundos regrese a su color

    1. Hacer un h3 con el texto: Esta es una etiqueta que cambia de color
    2. color por defecto azul
    3. hacer un botón que al pulsar cambie el color del texto
    4. cambia a rojo
    5. después de 2 segundos regrese a su color
        
    
      <template>
  <div>
    <h1>Bind</h1>
    <h1 :class="color">esta es una etiqueta que cambiará de color</h1>
    <button class="btn btn-primary" @click="cambio">CAMBIA</button>
  </div>
</template>
<!-- ************************************* -->
<script setup>
import { ref } from 'vue';

let color = ref('uno');

let cambio = () => {
  color.value = 'dos';
  setTimeout(() => {
    color.value = 'uno';
  }, 3000);
}

</script>
<!-- ************************************* -->
<style scoped>
.uno {
  background-color: antiquewhite;
  color: brown;
}
.dos {
  background-color: brown;
  color: aliceblue;
}
</style>

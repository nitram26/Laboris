IF

Teoría

  1. Conceptos Fundamentales

    En JavaScript, if es una estructura de control de flujo que permite ejecutar un bloque de código solo cuando se cumple una condición.
    Es la forma más básica de tomar decisiones en un programa.

    Fundamento

    Sintaxis:
    if (condicion) {
    // código que se ejecuta si la condición es verdadera
    }
    Condición: una expresión que se evalúa a truthy (verdadero) o falsy (falso).
    Puedes usar else para ejecutar un bloque alternativo si la condición es falsa:
    if (condicion) {
    // verdadero
    } else {
    // falso
    }
    También puedes usar else if para verificar múltiples condiciones en serie:
    if (condicion1) {
    // verdadero1
    } else if (condicion2) {
    // verdadero2
    } else {
    // ninguno de los anteriores
    }

    Notas útiles

    Valores truthy/falsy comunes:
    falsy: false, 0, "", null, undefined, NaN
    truthy: casi todo lo demás (incluye objetos, arrays, string vacía es falsy, así que "" no es truthy)
    Los operadores de comparación habituales (== vs ===) y los operadores lógicos (&&, ||, !) suelen combinarse dentro de la condición.

    Ejemplos rápidos

    if simple:
    if (edad >= 18) { console.log("Eres mayor de edad"); }

    con else:
    if (temperatura > 30) {
    console.log("Hace calor");
    } else {
    console.log("Hace fresco");
    }

    con else if:
    if (nota >= 90) {
    console.log("A");
    } else if (nota >= 80) {
    console.log("B");
    } else {
    console.log("Necesita mejorar");
    }

Ejercicios

  1. Escribe un programa que lea una edad (variable edad) y:
    si la edad es 18 o más, muestre "Bienvenido, puedes inscribirte".
    si es menor de 18, muestre "Lo siento, necesitas ser mayor de edad".

    const edad = 20;
    if (edad >= 18) {
    console.log("Bienvenido, puedes inscribirte");
    } else {
    console.log("Lo siento, necesitas ser mayor de edad"); }

  2. Dado un número n, usa if/else para:
    si n es positivo, imprime "positivo"; si es negativo, imprime "negativo"; si es 0, imprime "cero".

    const n = -5;
    if (n > 0) {
    console.log("positivo");
    } else if (n < 0) {
    console.log("negativo");
    } else { console.log("cero"); }

  3. Dada una variable contraseña y verifica si coincide con la cadena "secreto123".
    Si coincide: imprime "Acceso concedido".
    Si no coincide: imprime "Acceso denegado".

    const contrasena = "secreto123";
    if (contrasena === "secreto123") {
    console.log("Acceso concedido");
    } else {
    console.log("Acceso denegado"); }

  4. Dado una cantidad de productos cantidad:
    si cantidad >= 10, aplica un descuento del 15% e imprime el precio con descuento (asume precioUnitario = 9.99).
    si cantidad < 10, imprime el precio total sin descuento.

    const cantidad = 12;
    const precioUnitario = 9.99;
    if (cantidad >= 10) {
    const total = cantidad * precioUnitario;
    const conDescuento = total * 0.85;
    console.log("Total con descuento:", conDescuento.toFixed(2));
    } else {
    console.log("Total:", (cantidad * precioUnitario).toFixed(2)); }

  5. Dado una edad, imprime:

    "Infancia" si 0 <= edad < 12
    "Adolescencia" si 12 <= edad < 18
    "Adulto" si 18 <= edad < 65
    "Mayor" si edad >= 65
    Si la edad no es válida (menor que 0), imprime "Edad inválida".

    const edad = 25;
    if (edad < 0) {
    console.log("Edad inválida");
    } else if (edad < 12) {
    console.log("Infancia");
    } else if (edad < 18) {
    console.log("Adolescencia");
    } else if (edad < 65) {
    console.log("Adulto");
    } else {
    console.log("Mayor");

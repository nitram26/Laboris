VUE

    Para todos

  1. Tener instalado node
  2. Crear carpeta y llevarla a VSCode

    3. VITE

    Proceso

  3. npm create vite@latest
  4. Pedirá el NombreDelProyecto
  5. Pedirá iniciar con crear vite
  6. Seleccionar el framework
  7. Js o ts
  8. Cd nombreDelProyecto
  9. Npm install
  10. Npm run dev

VUE-CLI

  1. tener instalado node
  2. Tener instalado el CLI de Vue asi:
  3. npm install -g @vue/cli

    4. Proceso

  4. vue create nombreDelProyecto
  5. elegir manual o vue3
  6. npm add vuetify
  7. Proyecto terminado
  8. npm run serve
  9. Bootstrap necesita 2 instalaciones:
  10. npm install --save bootstrap
  11. npm install --save @popperjs/core

VUETIFY

  1. npm install vuetify
  2. npm install vite-plugin-vuetify

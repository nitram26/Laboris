VUE

    Para todos

  1. Tener instalado node
  2. Download
  3. cada cierto tiempo se debería actualizar el node hoy 180425
  4. Verificar la versión de Node.js
    node -v
    node --version
  5. Crear carpeta y llevarla a VSCode

    6. VITE

    Proceso

  6. Vite Guía
  7. npm create vite@latest
  8. Pedirá el NombreDelProyecto
  9. Pedirá iniciar con crear vite
  10. Seleccionar el framework
  11. Js o ts
  12. Cd nombreDelProyecto
  13. Npm install
  14. Npm run dev

VUE-CLI

  1. tener instalado node
  2. Tener instalado el CLI de Vue asi:
  3. npm install -g @vue/cli

    4. Proceso

  4. vue create nombreDelProyecto
  5. elegir manual o vue3
  6. npm add vuetify
  7. Proyecto terminado
  8. npm run serve
  9. Bootstrap necesita 2 instalaciones:
  10. npm install --save bootstrap
  11. npm install --save @popperjs/core

VUETIFY

  1. npm install vuetify
  2. npm install vite-plugin-vuetify

EL MAIN

        import { createApp } from 'vue'
        import './style.css'
        import App from './App.vue'
        import { createVuetify } from 'vuetify'
        import 'vuetify/styles'

        const vuetify = createVuetify()
        const app = createApp(App)
        app.use(vuetify)
        app.mount('#app')

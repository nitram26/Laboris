EL MAIN

        import { createApp } from 'vue'
        import './style.css'
        import App from './App.vue'
        import { createVuetify } from 'vuetify'
        import 'vuetify/styles'

        const vuetify = createVuetify()
        const app = createApp(App)
        app.use(vuetify)
        app.mount('#app')

VUE ROUTER

        INSTALL VUE-ROUTER
           
   PASO 1

              npm install vue-router
              SINO:
              npm install vue-router@4

                  
PASO 2

              CREAR CARPETA ROUTER Y INDEX.JS
              import { createRouter, createWebHistory } from 'vue-router'
              import HomeView from '../views/HomeView.vue'


              const routes = [
                {
                  path: '/',
                  name: 'Home',
                  component: HomeView
                },

              ]

              const router = createRouter({
                history: createWebHistory(),
                routes,
              })

              export default router


            
  PASO 3

              Importar el router en main.js
              Edita src/main.js para incluir el router:

              import { createApp } from 'vue'
              import App from './App.vue'
              import router from './router'

              const app = createApp(App)
              app.use(router)
              app.mount('#app')

            
  PASO 4

              Crear las vistas si no existen
              Ejemplo: src/views/HomeView.vue 
              Crear carpeta views y el archivo Homeview.vue


             
 PASO 5

              Asegúrate de que tu App.vue tenga  para mostrar las vistas según la ruta: Modifica tu App.vue para incluir la vista del router:


              <template>
                <div>
                  <nav>
                    <router-link to="/">Home</router-link> |
                  </nav>
                  <router-view />
                </div>
              </template>

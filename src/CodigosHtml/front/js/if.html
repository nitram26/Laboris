IF

Teoría

  1. Conceptos Fundamentales

    En JavaScript, if es una estructura de control de flujo que permite ejecutar un bloque de código solo cuando se cumple una condición.
    Es la forma más básica de tomar decisiones en un programa.

    Fundamento

    Sintaxis:
    if (condicion) {
    // código que se ejecuta si la condición es verdadera
    }
    Condición: una expresión que se evalúa a truthy (verdadero) o falsy (falso).
    Puedes usar else para ejecutar un bloque alternativo si la condición es falsa:
    if (condicion) {
    // verdadero
    } else {
    // falso
    }
    También puedes usar else if para verificar múltiples condiciones en serie:
    if (condicion1) {
    // verdadero1
    } else if (condicion2) {
    // verdadero2
    } else {
    // ninguno de los anteriores
    }

    Notas útiles

    Valores truthy/falsy comunes:
    falsy: false, 0, "", null, undefined, NaN
    truthy: casi todo lo demás (incluye objetos, arrays, string vacía es falsy, así que "" no es truthy)
    Los operadores de comparación habituales (== vs ===) y los operadores lógicos (&&, ||, !) suelen combinarse dentro de la condición.

    Ejemplos rápidos

    if simple:
    if (edad >= 18) { console.log("Eres mayor de edad"); }

    con else:
    if (temperatura > 30) {
    console.log("Hace calor");
    } else {
    console.log("Hace fresco");
    }

    con else if:
    if (nota >= 90) {
    console.log("A");
    } else if (nota >= 80) {
    console.log("B");
    } else {
    console.log("Necesita mejorar");
    }

Ejercicios

  1. Una atracción solo permite el acceso si la persona cumple **todas** estas condiciones:

    - Tiene al menos 12 años
    - Mide más de 1.40 metros
    - No tiene problemas cardíacos (`tieneProblemasCardiacos = false`)

    let edad = 13;
    let altura = 1.45;
    let tieneProblemasCardiacos = false;

    if (edad >= 12 && altura > 1.40 && !tieneProblemasCardiacos) {
    console.log("¡Puedes subir a la atracción!");
    } else {
    console.log("No cumples con los requisitos para subir."); }

  2. Escribe un programa que reciba una hora (en formato 24 horas) y determine si corresponde a:

    - Mañana (6 a 11)
    - Tarde (12 a 17)
    - Noche (18 a 23)
    - Madrugada (0 a 5)

    let hora = 14; if (hora >= 6 && hora <= 11) {
    console.log("Es mañana");
    } else if (hora >= 12 && hora <= 17) {
    console.log("Es tarde");
    } else if (hora >= 18 && hora <= 23) {
    console.log("Es noche");
    } else if (hora >= 0 && hora <= 5) {
    console.log("Es madrugada");
    } else {
    console.log("Hora inválida"); }

  3. Crea un programa que reciba una calificación (número entre 0 y 100) y muestre el tipo de nota según la siguiente escala:

    - 90 a 100: Excelente
    - 70 a 89: Aprobado
    - 50 a 69: Regular
    - Menos de 50: Reprobado

    let nota = 85;

    if (nota >= 90 && nota <= 100) {
    console.log("Excelente");
    } else if (nota >= 70 && nota < 90) {
    console.log("Aprobado");
    } else if (nota >= 50 && nota < 70) {
    console.log("Regular");
    } else if (nota >= 0 && nota < 50) {
    console.log("Reprobado");
    } else {
    console.log("Nota inválida");
    }

  4. Dado una cantidad de productos cantidad:
    si cantidad >= 10, aplica un descuento del 15% e imprime el precio con descuento (asume precioUnitario = 9.99).
    si cantidad < 10, imprime el precio total sin descuento.

    const cantidad = 12;
    const precioUnitario = 9.99;
    if (cantidad >= 10) {
    const total = cantidad * precioUnitario;
    const conDescuento = total * 0.85;
    console.log("Total con descuento:", conDescuento.toFixed(2));
    } else {
    console.log("Total:", (cantidad * precioUnitario).toFixed(2)); }

    "Adolescencia" si 12 <= edad < 18
    "Adulto" si 18 <= edad < 65
    "Mayor" si edad >= 65
    Si la edad no es válida (menor que 0), imprime "Edad inválida".

    const edad = 25;

    console.log("Edad inválida");
    } else if (edad < 12) {
    console.log("Infancia");

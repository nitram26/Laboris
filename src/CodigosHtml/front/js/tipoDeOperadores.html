TIPO DE OPERADORES

Aritméticos

  1. Teoría

    - +, -, *, /, %, ** (potencia)
    - Uso: operaciones numéricas básicas y potencias.
    - Nota: el operador + también realiza concatenación de cadenas si alguno de los operandos es una cadena.

  2. Buenas prácticas y consejos

    - Evita usar ==/!=; prefiere ===/!== para evitar coerción inesperada.
    - Usa el encadenamiento opcional y nullish coalescing para manejar valores ausentes de forma segura.
    - Sé consciente de la coerción de tipos en + (si una cadena está presente, podría convertir números a texto).
    - Mantén las expresiones lógicas simples; para condiciones complejas, usa funciones auxiliares para claridad.
    - Aprovecha el destructuring y spread para escribir código más limpio y mantenible.

Ejercicios

  1. Escribe una función que reciba dos números y retorne su suma, resta, multiplicación y división como un objeto {suma, resta, mult, div}.

    function operaciones(a, b) {
    return { suma: a + b, resta: a - b, mult: a * b, div: a / b }; }

  2. Calcula la potencia de un número base elevado a un exponente sin usar Math.pow.

    function potencia(base, exp) { return base ** exp; }

  3. Dada una hora en horas y minutos, devuelve los minutos totales desde la medianoche.

    function minutosTotales(h, m) { return h * 60 + m; }

  4. Parsea una cadena con formato "X+Y" y devuelve la suma.

    function parseSuma(str) {
    const [a, b] = str.split('+').map(Number);
    return a + b; }

  5. Calcula el residuo de dividir un número entre otro (mod).

    function residuo(a, b) { return a % b; }



Asignación

  1. Teoría

    - =, +=, -=, *=, /=, %=, **=
    - Uso: asignar o combinar valor actual con uno nuevo.
    - Ejemplos:
    - let a = 5;
    - a += 3; // a = 8

Ejercicios

  1. Dado una variable a = 5, aplica diferentes operadores de asignación para obtener a = 15.

  2. Usa += para combinar dos strings.

    s += " mundo"; // "Hola mundo"

  3. Usa operadores de asignación para intercambiar valores entre x e y sin usar temporales.

    let x = 2, y = 3;
    [x, y] = [y, x]; // x=3, y=2

  4. A partir de una nota numérica n, actualiza n sumando 10% (n += n * 0.1).

    let n = 70;
    n += n * 0.1; // 77

  5. Incrementa un contador solo si ya tiene valor (post-incremento).

    let c = 0;
    c++; // 1



Comparación

  1. Teoría

    - ==, ===, !=, !==, >, <, >=, <= -
    Diferencias clave: - == y !=
    realizan coerción de tipos (no recomendado en código moderno). -
    === y !== comparan valor y tipo sin coerción.
    - Consejos: usa siempre === y !== para evitar sorpresas.

  5. Devuelve true si un array tiene exactamente 3 elementos.

    function tresElementos(arr) { return Array.isArray(arr) && arr.length === 3; }

Operador de nullish coalescing

  1. Teoría

    - ??

    - Devuelve el operando de la derecha solo si el de la izquierda es null o undefined.
    - Ejemplo: const name = user?.name ?? "Invitado";

Ejercicios

  1. Dado dos enteros, devuelve su AND bit a bit.

    function yBit(a, b) { return a & b; }

  2. Invierte los bits de un entero de 8 bits.

    function invertir8(v) { return (~v) & 0xff; }

  3. Desplaza a la izquierda y derecha para obtener un nuevo valor.

    function desplazar(v) { return (v << 1) | (v >>> 7); }

  4. Comprueba si un número tiene un bit específico encendido (por ejemplo, bit 2).

    function tieneBit(n, pos) { return (n & (1 << pos)) !== 0; }

  5. Suma dos números sin usar el operador + (solo bit a bit).

    function sumaBit(a, b) {
    while (b !== 0) {
    const carry = a & b;
    a = a ^ b;
    b = carry << 1;
    }
    return a; }



Operador de uso de corchetes dinámicos (computed property)
  • Teoría

    En JavaScript, el concepto de “corchetes dinámicos” (computed properties) se refiere a la capacidad de definir propiedades de un objeto usando una expresión entre corchetes, de modo que el nombre de la propiedad se calcule en tiempo de ejecución. Esto se logra principalmente en dos contextos:
    Creación de objetos con propiedades computadas (computed property names)
    Se utiliza la sintaxis de corchetes [] dentro de la definición de un objeto.
    La expresión dentro de los corchetes se evalúa y su valor se usa como la clave de la propiedad.
    Útil para construir objetos dinámicamente o cuando el nombre de la propiedad depende de variables.

    Ejemplos:

    Usando una variable para el nombre de la propiedad
    const prop = "nombre";
    const obj = {
    [prop]: "Ana"
    };
    // obj = { nombre: "Ana" }

    Propiedades calculadas con expresiones
    const i = 1;
    const obj = {
    ["prop" + i]: "valor"
    };
    // obj = { prop1: "valor" }

    Acceso dinámico a propiedades (propiedad computada al acceder)
    Aunque comúnmente se habla de propiedades computadas al crear objetos, también es posible leer o escribir propiedades usando una clave calculada con bracket notation:
    const clave = "edad";
    const persona = { edad: 30 };
    persona[clave]; // 30
    persona[clave] = 31; // edad ahora es 31
    Notas útiles:

    Las corchetes dinámicas permiten nombres de propiedades que no son constantes, facilitando patrones como mapeos, tablas de hash simples, o construcción de objetos a partir de datos externos.
    En objetos literales, la sintaxis [expr] solo está disponible para propiedades computadas en ES6+.
    Ejemplos prácticos simples:

    Crear un objeto con propiedades cuyo nombre se toma de variables.
    Construir un contador de frecuencias: contar cuántas veces aparece cada palabra en un arreglo.
    Crear un objeto a partir de dos arreglos: claves y valores.

    • Ejercicios

    1. Crea un objeto persona donde la propiedad cuyo nombre está en la variable clave tenga un valor "Hola". Usa la variable clave = "saludo". const clave = "saludo";
      const persona = { [clave]: "Hola" };
      console.log(persona.saludo); // "Hola"
    2. Dadas las claves ['apple', 'banana', 'cherry'], crea un objeto dict donde cada clave tenga como valor su longitud (número de caracteres).

      const keys = ['apple', 'banana', 'cherry'];
      const dict = {};
      for (const k of keys) {
      dict[k] = k.length;

      } console.log(dict); // { apple: 5, banana: 6, cherry: 6 }

    3. Dada la lista de palabras ['manzana', 'manzana', 'pera', 'manzana', 'pera'], crea un objeto frec que cuente cuántas veces aparece cada palabra usando propiedades computadas.

      const palabras = ['manzana', 'manzana', 'pera', 'manzana', 'pera'];
      const frec = {};
      for (const w of palabras) {
      frec[w] = (frec[w] || 0) + 1;

      } console.log(frec); // { manzana: 3, pera: 2 }


    4. Dado un array de números numbers = [2, 4, 6], crea un objeto obj donde cada clave es "num" seguido del número (por ejemplo, "num2") y el valor es ese mismo número al cuadrado.

      const numbers = [2, 4, 6];
      const obj = {};
      for (const n of numbers) {
      obj[num${n}] = n * n;

      } console.log(obj); // { num2: 4, num4: 16, num6: 36 }

    5. Crea un objeto con propiedades dinámicas para los meses del año (enero, febrero, marzo) y asigna a cada una el número de días típico de ese mes (28, 28, 31). Usa un arreglo de nombres de meses y un bucle para construir el objeto con propiedades computadas.

      const meses = ["enero", "febrero", "marzo"];
      const dias = [31, 28, 31];
      const calendario = {};
      for (let i = 0; i < meses.length; i++) {
      calendario[meses[i]] = dias[i];

      } console.log(calendario); // { enero: 31, febrero: 28, marzo: 31 }

  5. Interpolación con valores booleanos

    Declara una variable booleana activa y usa interpolación para mostrar un estado basado en su valor (true / false).Usando un botón,mostrar un mensaje si es true y si es al revés mostrar otro mensaje. Con colores diferentes en cada caso. 

    <template>
<div class="bg-teal-lighten-2 text-white text-center py-5">
  <h1>ProbatioVue</h1>
  <h2 v-show="boolean == true" class="text-red">{{ mensaje1 }}</h2>
  <h2 v-show="boolean == false" class="text-purple">{{ mensaje2 }}</h2>
  <v-btn @click="boolean = !boolean" class="bg-blue"> Mensaje</v-btn>
</div>
</template>
<!-- +++++++++++++++++++++++++++++++++++++ -->
<script setup>
import { ref } from 'vue'

let boolean = ref(true)
let mensaje1 = ref('Hola, soy mensaje 1')
let mensaje2 = ref('Hola, soy mensaje 2')


</script>
<!-- +++++++++++++++++++++++++++++++++++++ -->
<style scoped></style>

  9. Interpolación con valores computados

    Crea un precio base y un porcentaje de descuento, luego usa una propiedad computada para calcular el precio final y mostrarlo en el template. 

    <template>
<div class="bg-teal-lighten-2 text-white text-center py-5">
  <h1>ProbatioVue</h1>
  <h2>La compra es de: $ {{ compra }}.00</h2>
  <h2>Su IGV 18% es: {{ igv }}.00</h2> <br>
  <h2>El pago mas IGV es: $ {{ precio }}.00</h2>
</div>
</template>
<!-- +++++++++++++++++++++++++++++++++++++ -->
<script setup>
import { computed, ref, } from 'vue';

let compra = 2500;
let igv = `${compra * 0.18}`;
let precio = ref(computed(() => {
  return compra * 1.18;
}));

</script>
<!-- +++++++++++++++++++++++++++++++++++++ -->
<style scoped></style>



  10. Interpolación dentro de atributos HTML

</div>
</template>
<!-- +++++++++++++++++++++++++++++++++++++ -->

</script>

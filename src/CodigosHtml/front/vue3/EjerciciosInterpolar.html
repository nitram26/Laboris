  2. Interpolación con variables dinámicas

    Crea una variable nombre, asigna un valor y muestra un saludo personalizado en el template usando interpolación. 

        <template>
        <div class="bg-teal-lighten-2 text-white text-center py-5">
          <h1>ProbatioVue</h1>
          <h2>{{ mensaje }}</h2>
        </div>
    </template>
    <!-- +++++++++++++++++++++++++++++++++++++ -->
    <script setup>
        let nombre = "Carlos M";
        let mensaje = "Hola " + nombre + ", bienvenido a ProbatioVue";
    </script>
      <!-- +++++++++++++++++++++++++++++++++++++ -->
    <style scoped></style>



  3. Interpolación con expresiones matemáticas

    Declara dos variables numéricas y utiliza interpolación para mostrar el resultado de una operación matemática en el template. 

      NOTA : LAS COMILLAS INVERTIDAS SE HACEN CON ALT+96 
<template>
  <div class="bg-teal-lighten-2 text-white text-center py-5">
    <h1>ProbatioVue</h1>
    <h2>{{ mensaje }}</h2>
  </div>
</template>
<!-- +++++++++++++++++++++++++++++++++++++ -->
<script setup>
  let num1 = 10;
  let num2 = 20;
  let mensaje = `El resultado de la suma ${num1} + ${num2} es: ${num1 + num2}`;
</script>
<!-- +++++++++++++++++++++++++++++++++++++ -->
<style scoped></style>

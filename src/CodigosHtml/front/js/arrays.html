Arrays

Teoría y Ejercicios

Ejercicios

  1. Dado un array de números, usa map para crear uno nuevo donde cada número esté multiplicado por 2.

    const original = [1, 2, 3, 4, 5];
    const dobles = original.map(num => num * 2);
    console.log(dobles); // [2, 4, 6, 8, 10]

  2. Usa filter para obtener sólo los números pares del array [3, 6, 9, 12, 15].

    const numeros = [3, 6, 9, 12, 15];
    const pares = numeros.filter(n => n % 2 === 0); console.log(pares); // [6, 12]

  3. Suma todos los números del array [10, 20, 30] usando reduce.

    const valores = [10, 20, 30];
    const suma = valores.reduce((acc, val) => acc + val, 0); console.log(suma); // 60

  4. Usa find para obtener el primer número mayor que 50 en [10, 55, 42, 67, 34].

    const numeros = [10, 55, 42, 67, 34];
    const mayorQueCincuenta = numeros.find(n => n > 50); console.log(mayorQueCincuenta); // 55

  5. Imprime cada elemento del array ['a', 'b', 'c'] en consola usando forEach.


    const letras = ['a', 'b', 'c'];
    letras.forEach(letra => console.log(letra)); // a // b // c

  6. Usa some para verificar si al menos una persona es mayor de 18. Array: [14, 16, 19, 12]

    :
    const edades = [14, 16, 19, 12];
    const hayMayor = edades.some(e => e >= 18); console.log(hayMayor); // true

  7. Usa every para comprobar si todos los valores en [21, 32, 25] son mayores de 18.



    const edades = [21, 32, 25];
    const todosMayores = edades.every(e => e >= 18); console.log(todosMayores); // true

  8. Ordena el array ['banana', 'apple', 'cherry'] alfabéticamente usando sort.

    const frutas = ['banana', 'apple', 'cherry'];
    const ordenadas = frutas.sort(); console.log(ordenadas); // ['apple', 'banana', 'cherry']

  9. Usa includes para comprobar si el número 5 está en [1, 3, 5, 7, 9].

    const numeros = [1, 3, 5, 7, 9];
    const contieneCinco = numeros.includes(5); console.log(contieneCinco); // true

  10. Usa slice para obtener los primeros 3 elementos del array [10, 20, 30, 40, 50].

    const lista = [10, 20, 30, 40, 50];
    const primerosTres = lista.slice(0, 3);
    console.log(primerosTres); // [10, 20, 30]


  11. Ejercicios con Métodos de Arrays (10)
    1. Método push()

    2. Método pop()
    Elimina el último elemento de un array de tareas pendientes y guarda ese valor en una variable.
    3. Método shift()
    Quita el primer estudiante de una lista de asistencia. Muestra quién fue eliminado.
    4. Método unshift()
    Agrega “Matemáticas” al inicio de un array de materias escolares.
    5. Método filter()
    Dado un array de números, crea uno nuevo que contenga solo los números mayores a 10.
    6. Método map()
    Transforma un array de precios en dólares multiplicándolos por 3.8 para obtener soles.
    7. Método reduce()
    Suma todos los valores de un array de calificaciones para obtener el total. Usa reduce().
    8. Método includes()

    9. Método sort()
    Ordena un array de nombres de forma alfabética. Después, imprímelo.
    10. Método forEach()

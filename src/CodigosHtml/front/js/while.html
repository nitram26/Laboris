  • Definición

    Un ciclo while en JavaScript es una estructura de control que repite un bloque de código siempre que se cumpla una condición. Se usa cuando no sabes cuántas iteraciones serán necesarias, pero sí quieres seguir ejecutando mientras una cierta condición sea verdadera.

  • Notas
    1. While significa mientras, es decir que se realiza mientras la condición sea verdadera

  2. Haz un juego en el que el usuario tenga que acertar un número al azar entre el 0 y el 9. Después el programa indicará cuántos intentos ha necesitado para acertar. 
      
    
// Juego: adivina un número entre 0 y 9
const numeroSecreto = Math.floor(Math.random() * 10); // 0..9
let intentos = 0;

while (true) {
  const entrada = prompt("Adivina el número entre 0 y 9:");

  // Si el usuario cancela, salimos del juego
  if (entrada === null) {
    alert("Juego cancelado.");
    break;
  }

  const numeroUsuario = Number(entrada);
  intentos++;

  // Validación básica
  if (!Number.isInteger(numeroUsuario) || numeroUsuario < 0 || numeroUsuario > 9) {
    alert("Por favor, introduce un número entero entre 0 y 9.");
    continue;
  }

  if (numeroUsuario === numeroSecreto) {
    alert(`¡Correcto! Has acertado en ${intentos} intento(s).`);
    break;
  } else if (numeroUsuario < numeroSecreto) {
    alert("Más alto.");
  } else {
    alert("Más bajo.");
  }
}
